Smart Resume Shortlisting System

Overview

The Smart Resume Shortlisting System automatically analyzes resumes and compares them with job requirements. It helps recruiters quickly shortlist the most relevant candidates.

Technologies Used

- Python
- Flask
- HTML
- CSS
- PyPDF2

Features

- Upload multiple resumes
- Extract text from PDF resumes
- Compare resumes with required skills
- Automatically shortlist candidates

How It Works

1. User uploads multiple resumes.
2. The system extracts text from resumes using PyPDF2.
3. Skills are compared with job requirements.
4. Candidates are categorized as Shortlisted or Rejected.
